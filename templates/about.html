About CropGuard AI

🎯 Project Goal

CropGuard AI is an intelligent system that helps farmers identify crop diseases from leaf images using deep learning and provides expert advisory through Generative AI, helping reduce agricultural losses in rural and remote areas.

🧠 Technology Stack

  • 🔬 Model: MobileNetV2 (PlantVillage, 38 classes)
  • 🤖 Advisory: Google Gemini 1.5 Flash
  • 🌐 Backend: Python Flask
  • 📊 Dataset: PlantVillage (50,000+ images)
  • 📄 Reports: ReportLab PDF

🌱 Supported Crops

Apple, Blueberry, Cherry, Corn, Grape, Orange, Peach, Bell Pepper, Potato, Raspberry, Soybean, Squash, Strawberry, Tomato and more.

👥 Group 17

This project was developed as part of the Agriculture Technology domain project — Crop Disease Detection and Advisory System.
